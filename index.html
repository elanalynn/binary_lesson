What is a bit?

A bit (short for binary digit) is the smallest unit of data in a computer. A bit has a single binary value, either 0 or 1.

ASCII

What is ASCII?

ANSI (American National Standards Institute) created ASCII (American Standard Code for Information Exchange) in 1963 to encode characters into a standard language that computers could understand.

ASCII does not cover a large number of special characters.

UTF-8

UTF-8

UTF-8 is a compromise character encoding that can be as compact as ASCII (if the file is just plain English text) but can also contain any unicode characters (with some increase in file size). UTF stands for Unicode Transformation Format. The '8' means it uses 8-bit blocks to represent a character.

87% of all web sites use UTF-8
